## Phase 5 design decisions

| Date | Decision | Reason |
|---|---|---|
| July 16, 2026 | Interpret only the root `.gitignore` in Phase 5 | Preserve deterministic documented MVP scope without claiming full Git parity |
| July 16, 2026 | Keep mandatory exclusions separate from repository rules | Prevent negation patterns from re-enabling sensitive or generated paths |
| July 16, 2026 | Verify descriptor identity before and after bounded reads | Reduce discovery-to-read races while enforcing the byte limit after open |
| July 16, 2026 | Build in a restricted sibling and verify before promotion | Keep an existing healthy live index untouched until a complete candidate is proven usable |
| July 16, 2026 | Close every Chroma client before directory replacement | Avoid stale handles and make rollback/promotion portable across tested platforms |

| July 16, 2026 | Enforce directory entry limits before sorting | Prevent a wide attacker-controlled directory from bypassing intended memory bounds during materialization |
| July 16, 2026 | Revalidate authoritative index status before every engine operation | Prevent missing, corrupt, stale, or fingerprint-inconsistent indexes from being queried as healthy |
| July 16, 2026 | Keep symbol metadata uncached per engine instance | Avoid hiding a promoted index replacement or serving stale symbol state |
| July 16, 2026 | Bound query text, symbol names, snippets, and returned match counts | Limit local memory use and public source exposure without adding a dependency or configuration field |
| July 16, 2026 | Use deterministic semantic and five-group symbol ordering | Make identical index and request inputs produce stable evidence across calls |
| July 16, 2026 | Keep CLI help and version free of runtime dependencies | Make discovery reliable without configuration, model, storage, or network side effects |
| July 16, 2026 | Escape JSON to an ASCII-safe deterministic representation | Preserve exact parsed values without literal terminal separators, bidi controls, or surrogates on stdout |
| July 16, 2026 | Revalidate the exact configured runtime immediately before reset deletion | Keep the destructive sink narrow and prevent caller-selected targets from reaching it |

## Important decisions

| Date | Decision | Reason |
|---|---|---|
| July 14, 2026 | Use Python 3.12 | Required Build Week baseline |
| July 14, 2026 | Use MCP 1.x with `<2` bound | Avoid pre-release MCP v2 APIs |
| July 14, 2026 | Limit MVP to Python | Protect MVP scope and deadline |
| July 15, 2026 | Resolve config paths relative to the TOML file | Deterministic behavior independent of invocation directory |
| July 15, 2026 | Keep CLI root overrides immutable and invocation-relative | Support future CLI use without mutating stored configuration |
| July 15, 2026 | Use a defensive `MappingProxyType` copy for language counts | Provide genuine nested immutability while preserving JSON-object output |
| July 15, 2026 | Reject every symlink/junction component in a configuration path | Prevent ancestor-symlink escapes before binary TOML loading |
| July 15, 2026 | Make reset-target validation exact and non-destructive | Prevent repository-root or outside-root deletion in future phases |
| July 15, 2026 | Keep parsing bytes-in and filesystem-free | Preserve the path-guard boundary and avoid parser-side reads |
| July 15, 2026 | Extract only module definitions and direct methods | Match Phase 2 scope and avoid nested-symbol ambiguity |
| July 15, 2026 | Use token-aware signature normalization | Preserve literal contents while producing deterministic single-line signatures |
| July 15, 2026 | Precompute signature line offsets | Avoid quadratic prefix scanning on large multiline declarations |
